# Emotion-Onomatopoeia-Dictionary


## 概要
DOMを用いたウェブページ

## 説明
このプロジェクトは、感情と擬音語を表示する 1 ページアプリケーションです。

カテゴリをクリックすることで、それに関連する感情の擬音語のカードが表示されるセクションにジャンプします。
